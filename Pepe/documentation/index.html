1. Basic configuration

If you are happy with the web page as we organized it, there are only few steps you need to do before going online.

1.1 - Connect your Disqus account

For comments, Ghost uses external services such as Disqus. Pepe already contains the necessary code to embed Disqus comments, you only need to change the account.
Go to default.hbs on line 41 and replace pepe-muvolab with your site short name of your, after you have registered.

FILE default.hbs - line 41

<body class="{{body_class}}" data-disqus='pepe-muvolab'>

1.2 - Choose your style

You can use Pepe in its basic version or customize it, you just need to add the reference to its css.
Open default.hbs at line 20 and remove the comment from one or more of the link tag. You can choose whatever combination you like, here some examples

1.3 - Edit the menu

Ghost 0.4.~ supports static pages, but does not provide a system to manage the menu.
For this reason you will need to edit the menu directly in the theme:
Simply open the file partials/menu.hbs and add the items that you need to ul, using as href the URL to the page (taken from the Ghost admin panel).
